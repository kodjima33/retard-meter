How it works

This app captures system audio and transcribes speech in real-time.

Select your audio device from the dropdown menu, then click "Start Recording" to begin capturing and transcribing audio.

Device Selection

To capture microphone audio: select your preferred microphone from the dropdown.

To capture system audio: install BlackHole audio driver (via brew install blackhole-2ch), set up multi-output in Audio MIDI Setup, and select "BlackHole 2ch" in the dropdown.

Requirements

You need to:

  • Grant microphone permissions
  • Install SoX (brew install sox)
  • Set up Google Speech API credentials
